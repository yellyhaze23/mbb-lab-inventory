# GMBD MBB Lab Inventory System

Web-based laboratory inventory platform for managing chemicals, consumables, stock operations, usage logs, and student PIN-based usage workflows.

## Quick Start
```bash
npm install
npm run dev
```

Build checks:
```bash
npm run lint
npm run build
```
